# AI SQL Query Generator

An AI-powered web application that converts natural language into SQL queries using Prompt Engineering and Google Gemini.

## Features

- Generate SQL from plain English
- Supports multiple SQL dialects
- SQL explanations
- Schema-aware query generation
- Copy generated SQL
- Modern responsive interface

## Tech Stack

- Python
- FastAPI
- HTML
- CSS
- JavaScript
- Google Gemini API
- Prompt Engineering
- Pydantic
- Requests

## How it Works

1. User enters a request.
2. FastAPI receives the request.
3. A prompt is created.
4. Google Gemini generates SQL.
5. The SQL and explanation are displayed.

## Installation

```bash
pip install -r requirements.txt
python app.py
```
